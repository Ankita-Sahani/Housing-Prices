# Housing-Prices
Housing Price Prediction using Machine Learning. This project analyzes housing data and predicts property prices using Linear Regression and Random Forest models. It includes data preprocessing, feature engineering, visualization, and model evaluation using MAE, RMSE, and R² metrics to identify key factors influencing house prices.

# Dataset 
Link - https://www.kaggle.com/datasets/yasserh/housing-prices-dataset
The Housing dataset provides both numerical and categorical information about houses. Features such as area, number of bedrooms, bathrooms, parking availability, furnishing status, and location preferences significantly influence house prices, making this dataset suitable for real estate price prediction and regression-based machine learning projects.
